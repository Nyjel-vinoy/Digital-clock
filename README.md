# Digital Clock ⏰

A simple **Digital Clock** that displays **hours, minutes, and seconds** using **HTML, CSS, and JavaScript**.

## 🛠 Technologies Used
- **HTML** – Structure of the clock  
- **CSS** – Styling and layout  
- **JavaScript** – Dynamic time update
